This subroutine performs the eigen decomposition of a given matrix. It is a binding to LAPACK's dgeev function for eigen decomposition. For more details, see LAPACK's reference: LAPACK's dgeev function documentation for eigen decomposition.

Parameters
[in]nthe order of the given matrix
[in,out]Ainput matrix of dimensions (n,n), on output it gets overwritten
[out]Woutput vector (allocated on input) to store the eigenvalues
[out]Voutput matrix (allocated on input) to store the eigenvectors
[out]infooutput status: if 0 then successful exit
